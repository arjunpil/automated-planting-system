# Automated Planting System

## Overview
An automated seed planting system built using an Arduino Mega 2560 and MG996R motor. 
The system uses a linear actuator mechanism to precisely control seed funneling and 
release into fertilized soil.

## Hardware Components
- Arduino Mega 2560
- MG996R High Torque Servo 
- CAD Linear Actuator 
- HC-SR04 Sensor
- HC-05 Bluetooth Module
- Carbon Fiber Base w/ treads 

## How It Works
- Linear actuator controls precise seed funneling and release
- Motion sensors monitor and regulate planting depth in real time
- Bluetooth communication enables wireless control of tank movement
- All control logic written in C++

## Technologies Used
- C++
- Arduino IDE
- Bluetooth Communication Protocol w/ Bluetooth Arduino App
